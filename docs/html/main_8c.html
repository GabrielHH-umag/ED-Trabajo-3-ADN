Descripción detallada

Punto de entrada del Analizador de ADN basado en estructuras Trie.

Este archivo inicializa la interfaz de comandos (CLI) del programa, la cual permite ejecutar:

  • bio start m
  • bio read archivo.txt
  • bio search GEN
  • bio all
  • bio max
  • bio min
  • bio exit

La lógica principal y las funciones internas se encuentran en los módulos bio_commands.c, bio_func.c y bio_struct.h. Este archivo únicamente inicia el flujo de ejecución.
