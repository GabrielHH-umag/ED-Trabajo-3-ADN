Descripción detallada

Declaraciones del módulo encargado de la interfaz de comandos (CLI) para el Analizador de ADN basado en estructuras Trie.

Este módulo implementa el manejo de entrada del usuario, el procesamiento de comandos y la vinculación con las funciones lógicas internas del analizador de ADN. Los comandos disponibles permiten iniciar el árbol, cargar una secuencia genética desde archivo, buscar genes, listar todos los existentes y calcular los de mayor o menor frecuencia.
