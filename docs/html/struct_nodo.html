Descripción detallada

Representa un nodo del Trie utilizado para almacenar genes.

Cada nodo posee exactamente 4 hijos, uno por cada base nitrogenada:

  • 〈0〉 → A
  • 〈1〉 → C
  • 〈2〉 → G
  • 〈3〉 → T

Los nodos hoja (nivel == profundidad m) almacenan dinámicamente las posiciones dentro de la secuencia S donde aparece el gen correspondiente.
