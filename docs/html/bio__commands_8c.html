Descripción detallada

Implementación del módulo encargado de la interfaz de comandos (CLI) para el analizador de ADN basado en Trie.

Este archivo contiene:

  • El bucle interactivo de lectura de comandos.
  • La lógica de interpretación de argumentos.
  • La vinculación de cada comando con las funciones lógicas del Trie.
  • Funciones de búsqueda, listados e inspecciones (all, max, min).

Todos los comandos siguen el formato: bio <accion> <argumento>

Ejemplo: bio start 3 bio read adn.txt bio search ACT bio max
